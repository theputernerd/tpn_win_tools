# Project Conventions

Repo-specific conventions that are too specific for `global-conventions.md` but
apply across many tasks. **Put your customisations here, not in `AGENTS.md`** —
`AGENTS.md` is framework-managed and gets overwritten on `--upgrade`; this
directory is user content and is never touched.

Create one: `instructions/helpers/create-project-convention.sh <slug>`
